### 📌 Portfolio Website

This is a simple personal portfolio website built using **HTML** and **CSS**. It consists of two main pages is a "Home page" and a "Contact page", designed with a clean and minimal layout.

The project focuses on practicing core front-end development skills by creating a structured and visually appealing website without using any frameworks or JavaScript.

### 🚀 Features

* Clean and minimal design
* Two-page structure (Home & Contact)
* Easy navigation between pages
* Responsive layout for different devices

### 🛠️ Technologies Used

* HTML5
* CSS3

### 📄 Pages Included

* Home Page – Introduction and basic information
* Contact Page – Simple contact details or form

### 📖 What I Learned

* Creating multi-page websites using HTML
* Linking pages with navigation
* Styling layouts using CSS
* Building a responsive design from scratch

### 📬 Future Improvements

* Add more pages (Projects, About, Skills)
* Include JavaScript for interactivity
* Improve UI with animations and transitions
